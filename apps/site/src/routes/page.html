KuratchiJS for LLM-driven development.

Cloudflare Workers-native framework with file-based routes, server actions, Durable Object RPC, and batteries-included auth/ORM integrations.

  • Primary docs for models: /docs
  • Machine index: /_assets/llms.txt
  • Machine schema: /_assets/llms.json

Core model

HTML-first routes in src/routes, compiled to .kuratchi worker output.

Progressive enhancement via declarative attributes and server actions.

Batteries included

Config-driven ORM, auth, and Durable Objects from kuratchi.config.ts.

Cloudflare-first defaults with Wrangler-aligned deployment.
