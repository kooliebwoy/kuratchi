## Framework build model

`kuratchi build` scans `src/routes/` and generates a stable worker entrypoint at `.kuratchi/worker.ts`.

```json
{
  "main": ".kuratchi/worker.ts"
}
```

That lets application code stay focused on routes and server modules while the compiler owns the generated worker plumbing.
